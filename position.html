In static- by default all positions are using position static. It will not support top/bottom/left/Right It can be only moved with the margin property only.

In relative - the postion of element is in the flow of document like static. it changes itss position where it normally appears. It allows to adjust top/bottom/left/right and also move with margin etc Relative takes its own space

Abosulte - the position of element does not change place from its normal position it will always change in the reference of its parent only. it does not leaves a gap on a normal position The purpose of absolute position is for overlapping the elements
